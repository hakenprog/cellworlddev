== Description ==
[Wow Divi Carousel Lite](https://wowcarousel.com/) is a Divi touch-enabled plugin that lets you create a beautiful responsive slider or carousel. 
 
### **3 FREE PREMIUM-LIKE MODULES AND MORE ON ITS WAY**
Unlike most other Divi plugins, Wow Carousel offers itself with many robust free modules.

- [**Image Carousel**](https://wowcarousel.com/modules/image-carousel/) – Create interactive image and text carousels using our image carousel module which comes with a lot of options.
- [**Logo Carousel**](https://wowcarousel.com/modules/logo-carousel/) – Display your clients or products using our logo carousel module, and display these items with styles.
- [**Twitter Feed Carousel**](https://wowcarousel.com/modules/twitter-feed/) – Display your awesome team decorating in the Twitter feed mode applying various styles, texts, images, and social links.

== Installation ==
Note: This plugin works with Divi Theme or Divi Builder. Make sure you have Divi Theme or Divi Builder installed.

1. Upload the plugin folder to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress
